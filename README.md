# 🚀 SkillSync Landing Page

A modern and responsive Course Hub Landing Page built using React and Tailwind CSS. This project focuses on clean UI design, smooth user experience, and mobile responsiveness.

---

## 🔗 Live Demo
👉 https://skill-sync-landing-page.vercel.app/

---

## 💻 GitHub Repository
👉 https://github.com/chiragha/SkillSync-Landing-Page

---

## ✨ Features

- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Clean and Modern UI
- ⚡ Fast Performance
- 🎯 Well-structured Sections (Navbar, Hero, Courses, CTA)
- ✨ Basic UI Animations for better user experience

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)

- npm install
